# AirSpeed

一个无需构建工具的浏览器网络测速页面，使用 Cloudflare 测速端点测量延迟、下载和上传速度，并根据下载结果估算对应的常见宽带档位。

标准模式会先预热连接，再使用 4 个并发数据流进行最长 6 秒的下载和上传采样，同时显示稳定速度和峰值速度。为避免无限消耗流量，标准模式最多使用约 760 MB，省流模式最多使用约 13 MB。

## 使用

直接打开 `index.html` 即可。部分浏览器可能限制从本地文件发起网络请求，此时可在该目录启动附带的本地预览服务器：

```powershell
.\serve.ps1
```

然后访问 `http://localhost:8080`。
